* 설치

  apt update

  apt install bind9 bind9utils bind9-doc -y

  systemctl start named

  systemctl enable named

-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
* 패키지 설치 시 기본 설치 경로

  /etc/bind
-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
* Ubuntu에서 bind9 패키지를 설치하면 기본으로 자동 생성되는 AppArmor 정책 파일에 따라 권한 문제가 발생 할 수 있기 때문에 파일에 경로에 대한 권한을 추가해야 함 ( 파일 수정 후 적용까지 )

  vi /etc/apparmor.d/usr.sbin.named

  /etc/bind/** rw, 

  /var/lib/bind/** rw,

  /var/lib/bind/ rw,

  /var/cache/bind/** lrw,

  /var/cache/bind/ rw,

  /etc/bind/zone/** rwk, ==>  추가

  /logs/named/** rw, ==> 추가

  /logs/named/ rw,  ==> 추가
-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
* 추가 후 적용
  
  apparmor_parser -r /etc/apparmor.d/usr.sbin.named

  systemctl restart named
